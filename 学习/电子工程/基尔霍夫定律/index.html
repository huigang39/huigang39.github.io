# 基尔霍夫定律

# 概述

基尔霍夫定律包括基尔霍夫电流定律（KCL）和基尔霍夫电压定律（KVL）。

  1. 基尔霍夫电流定律（KCL）：确定连接在同一结点上各支路电流的关系；
  2. 基尔霍夫电压定律（KVL）：确定回路中各部分电压之间的关系。

# 基尔霍夫电流定律（KCL）

# 定义

在任一时刻，流入某一结点的电流之和，必定等于流出该结点的电流之和。

# 依据

电荷守恒定律：由于电路中的连接点、结点，既不能产生电荷，也不能消耗电荷。因此，在任一时间段，电路中的任一结点，有多少电荷流入该结点，必定就有多少电荷流出该结点，结点上不会有电荷的堆积。

也就是说，在电路中，任一时刻，一个结点上电流的代数和恒等于 0。

# 使用

先确定参考方向，通常取流入结点的电流为正号（+），流出结点的电流为负号（-）。KCL 不仅适用于电路中的任一结点，也可推广应用于电路中任一假设的闭合面，也称广义结点。即流入任一闭合面的电流之和，等于该闭合面流出的电流之和。

# 基尔霍夫电压定律（KVL）

# 定义

从回路中任意一点出发，沿回路循行一周（顺时或逆时），电位升之和必定等于电位降之和。

# 依据

能量守恒定律：如果使电荷沿着闭合的回路移动一周，则电荷获得和失去的净能量为 0，电压可以表示势能。

沿任一回路循行方向，回路中各段电压的代数和恒等于 0。

# 使用

若电压降，取正号（+）；电压升，取负号（-）。

若电位升，取正号（+）；电位降，取负号（-）。

KVL 不仅适用于回路，而且可以推广应用于开口电路。
