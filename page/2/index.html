# 存储介质介绍 存储介质（存储器）被广泛应用在包括但不限于单片机、通用计算机等领域，作为电子设备中用来保存信息的记忆（Memory）设备，它具备存储数据和信息的能力，并且可以连续执行程序，进行广泛的信息处理。 在数字系统中，只要可以保存二进制数据的都可以称为存储器； 在集成电路中，具有存储功能的电子元器件也成为存储器（如 RAM、FIFO）； 在系统中，具有实物形式的存储设备也叫存储器，如内存条、内存卡等。
